🦾 AURA SYNC: The Ghost Protocol
Aura Sync is a futuristic, high-intensity fitness dashboard designed for the modern athlete. Moving away from standard "log" apps, Aura Sync utilizes a HUD-inspired interface to track real-time performance, muscle evolution, and nutritional intake through the "Ghost Protocol."

🚀 Key Features
The Ghost Protocol Timer: A chained, auto-transitioning performance clock. It handles the 1-minute Time Under Tension (TUT) lifting phase and automatically triggers the 90-second recovery phase, ensuring zero downtime during EOS sessions.

Evolution Rank Engine: An RPG-inspired muscle heatmap. Using a 5-tier color system (from The Void to Aura God-Tier), your dashboard physically evolves as you log volume and intensity.

Fuel Vision AI: A dedicated nutrition terminal featuring a "Snap to Log" scanner and a persistent meal-planning notepad for tracking high-protein protocols.

Neural Latency (Sync): Real-time monitoring of HRV and Sleep metrics. The interface remains "locked" until a wearable handshake is detected, preventing ghost data from skewing your recovery stats.

EOS-Grade Machine Library: A comprehensive database of over 40+ commercial-grade machines, categorized for Upper, Lower, Cardio, and Functional training.

🛠️ Technical Stack
Framework: React + Vite

Styling: Tailwind CSS (Neon-Dystopian Theme)

Persistence: LocalStorage API for offline-first gym floor reliability.

Deployment: Vercel (CI/CD via GitHub)

APIs: Web Bluetooth (for wearable sync) & MediaDevices (for Fuel Vision scanning).
